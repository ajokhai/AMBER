# Amber Mobile App

This is the mobile application for Amber, built with Flutter.

## Getting Started

### Prerequisites

- Flutter SDK
- Android Studio / Xcode (for emulators/simulators)

### Running the App

1.  Navigate to the `mobile` directory:
    ```bash
    cd mobile
    ```

2.  Get dependencies:
    ```bash
    flutter pub get
    ```

3.  Run the app:
    ```bash
    flutter run
    ```

## Project Structure

- `lib/`: Contains the Dart code for the application.
- `test/`: Contains unit and widget tests.
